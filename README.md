An automated tool that uses **Google Gemini 2.5 Flash** to transfer gemstone appearances onto ring models. This project allows users to batch-process multiple ring designs with various gemstone references to create photorealistic product images.

## âœ¨ Features

- **Folder Upload Support:** Drag and drop entire folders of ring models and stone references.
- **Batch Processing:** Automatically generates all possible combinations of rings and stones.
- **High Resolution:** Outputs photorealistic 1024x1024 studio-quality images.
- **Flexible Input:** Supports both local file uploads and image URLs.
- **Customizable Prompts:** Fine-tune the AI output with custom instructions.
- **Real-time Progress:** Track generation progress through a dedicated Gradio interface.


## ğŸ“Š How it Works

The system utilizes the **Gemini 2.5 Flash Image** model. It takes two images as input:
1. **Image 1 (Ring Model):** The structural base.
2. **Image 2 (Stone Reference):** The color and pattern source.

The AI performs a style transfer, preserving the ring's geometry while updating the center stone to match the reference, all while maintaining a consistent photorealistic studio aesthetic on a pure white background.
